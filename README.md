# ☕ Coffee Ordering App

A modern Android coffee ordering application built with **Kotlin** and **Jetpack Compose**. The app provides an intuitive user interface for browsing coffee products, viewing product details, managing favourites, adding items to a cart, and selecting payment methods.

## ✨ Features

- Coffee catalogue with product images, descriptions, and prices
- Detailed product view with coffee size selection
- Add-to-cart functionality
- Cart management with quantity controls
- Favourites section for saved coffee products
- Payment method selection with Cash, Online, and Card options
- User profile screen
- Bottom navigation across Home, Cart, Favourites, and Profile
- Reusable UI components built with Jetpack Compose
- Custom coffee-themed typography, colours, and layouts

## 🛠️ Tech Stack

- **Language:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Navigation:** Navigation Compose
- **IDE:** Android Studio
- **Version Control:** Git and GitHub

## 📲 App Flow

```text
Welcome Screen
      ↓
Home Screen
      ↓
Product Details
      ↓
Add to Cart
      ↓
Cart & Quantity Management
      ↓
Payment Method Selection
```

Users can also navigate between **Home**, **Cart**, **Favourites**, and **Profile** using the bottom navigation bar.

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- Android Studio
- Android SDK
- Git

### Installation

1. Clone the repository:

```bash
git clone https://github.com/cipherByte7/CoffeeApp.git
```

2. Open the project in Android Studio.

3. Allow Gradle to sync and download the required dependencies.

4. Run the application on an Android emulator or a physical Android device.
